# 妖境說

HTML playable build for GitHub Pages.

## How to play locally
Open `index.html` in a browser.

If audio or browser security behaves differently when double-clicking, run a small local server:

```powershell
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## GitHub Pages
Upload this folder's contents to a GitHub repository root:

- `index.html`
- `style.css`
- `game.js`
- `assets/`

Then enable GitHub Pages for the repository.
